# DML (Data Manipulation Language)

Conjunto de comandos SQL para que o usuário interaja com os dados armazenados no BD. Por exemplo:

### Select 

Efetua leitura de dados (consultas) nas tabelas do BD. Efetua as sete operações da álgebra relacional. “Select <colunas> from <tabela> where <condição>”

### Insert 

Inclui registros nas tabelas do BD. Insert into <tabela> values <valores>

### Update 

Altera dados nas tabelas do BD. Update <tabela> set <campo>=<novo valor> where <condição>

### Delete 

Exclui registros das tabelas do BD. Delete  <tabela> where <condição>
