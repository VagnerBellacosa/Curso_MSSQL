# O que é DML | DDL | DCL | DQL 

## Linguagem SQL te apresento nesse artigo e vídeo o significado das siglas DML, DDL, DCL e DQL tão importantes quando utilizamos a Linguagem SQL.

### SQL-O-que-é-DDL-DML-DCL

Traduzindo o que significa cada Sigla:

#### DML 
 Data Manipulation Language: para representar o conjunto de instruções da linguagem SQL para manipular os dados.

#### DDL
DDL conjunto de instruções e comandos para definição de dados.

#### DCL 
DCL instruções para controlar as permissões de acesso e uso aos objetos do banco de dados.

#### DQL 
DQL representando a principal instrução SQL, o SELECT, comando que realiza consultas no SGBD(Sistema de Gerenciamento de Banco de Dados).

O SQL é uma linguagem que representa um conjunto de comandos estruturados para consulta e manipulação dos dados no banco de dados, seja ORACLE, SQL Server, DB2 e outros.
